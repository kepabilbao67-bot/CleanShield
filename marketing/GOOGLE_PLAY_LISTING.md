## Declaración de Permisos

### Servicio VPN (android.net.VpnService)
**Justificación:** CleanShield utiliza el servicio VPN de Android exclusivamente para crear una conexión local que filtra consultas DNS. No se transmite, almacena ni intercepta tráfico de red. La VPN no se conecta a ningún servidor remoto. Todo el procesamiento ocurre en el dispositivo del usuario.

### Servicio de Accesibilidad (AccessibilityService)
**Justificación:** Se utiliza exclusivamente para detectar y bloquear contenido explícito en aplicaciones de terceros (navegadores y redes sociales). El servicio solo lee nombres de paquetes y URLs visibles para compararlos con listas de bloqueo locales. No recopila, almacena ni transmite texto, contraseñas ni datos personales del usuario.

### Administrador del Dispositivo (DeviceAdminReceiver)
**Justificación:** Permiso opcional que el usuario puede activar para prevenir la desinstalación no autorizada de la aplicación (modo de protección máxima). Se activa únicamente con consentimiento explícito del usuario y puede ser desactivado en cualquier momento siguiendo el procedimiento de seguridad configurado por el propio usuario.

### Ejecución en Primer Plano (FOREGROUND_SERVICE)
**Justificación:** Necesario para mantener activo el servicio de filtrado DNS mientras el dispositivo está en uso. Se muestra notificación persistente al usuario indicando que la protección está activa.

### Inicio en Boot (RECEIVE_BOOT_COMPLETED)
**Justificación:** Permite que la protección se reactive automáticamente al reiniciar el dispositivo, garantizando cobertura continua sin intervención manual.

## Sección de Seguridad de Datos

### ¿La app recopila datos?
Sí (opcionalmente)

### Datos recopilados (solo con modo tutor activado):
- **Actividad en la app:** Estadísticas agregadas de uso (bloqueos diarios, racha)
- **Identificadores:** UUID aleatorio del dispositivo

### Datos NO recopilados:
- Ubicación
- Información personal (nombre, email solo para auth)
- Información financiera
- Historial de navegación
- Contactos
- Fotos/vídeos
- Archivos

### Prácticas de seguridad:
- Los datos se cifran en tránsito (TLS 1.3)
- Los datos se cifran en reposo (AES-256)
- Puedes solicitar la eliminación de datos
- Los datos no se comparten con terceros
